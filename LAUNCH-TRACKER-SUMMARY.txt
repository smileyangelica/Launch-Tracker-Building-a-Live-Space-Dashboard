TECHNICAL SPECIFICATIONS
───────────────────────────────────────────────────────────────────────────────

ARCHITECTURE:
  • Frontend: HTML5 + CSS3 + Vanilla JavaScript
  • Data Source: SpaceX API v4
  • Deployment: GitHub Pages (static hosting)
  • Dependencies: Zero external dependencies

PERFORMANCE:
  • Page Load: < 2 seconds (typical)
  • File Size: 29.8 KB (complete, single file)
  • API Response: < 500ms (typical)
  • Memory Usage: < 5MB
  • Countdown Accuracy: ±1 second

BROWSER SUPPORT:
  • Chrome 60+
  • Firefox 55+
  • Safari 12+
  • Edge 79+
  • Mobile browsers (iOS Safari, Chrome Mobile)

DATA SOURCES:
  • SpaceX API: https://api.spacexdata.com/v4
  • Endpoints: /launches/upcoming, /rockets/{id}, /launchpads/{id}
  • Rate Limit: None (public endpoint)
  • Authentication: Not required
  • Response Format: JSON
